# Workflow sugerido

## Com Copilot

Use o Copilot para:

- explorar a ideia inicial
- refinar perguntas
- revisar o texto gerado
- transformar o pitch em slides e assets complementares

## Com OpenAI

Use a API para:

- gerar o startup pack inicial
- padronizar a estrutura de saida
- automatizar o pipeline em app ou backend

## Fluxo combinado

1. Escreva a ideia bruta.
2. Gere o startup pack.
3. Revise o material com Copilot.
4. Ajuste tom, mercado e proposta de valor.
5. Converta em apresentacao, email e identidade visual.
